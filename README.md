Онлайн Чат

Технологии:
Backend: ASP.NET Core, EF Core, MS SQL + SignalR;
Frontend: JavaScript, HTML, CSS;

Функции:
 - обмен сообщениями в реальном времени (SignalR)
 - сегментная подгрузка истории сообщений

Инструкции по запуску в Docker
1. Клонировать репозиторий: https://github.com/frost945/TestChatSignalR.git
2. docker compose up --build

Доступ к приложению
Frontend: http://localhost:5000
